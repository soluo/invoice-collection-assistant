## Project Summary

**Relance Factures** is a French invoice collection management application built for artisans, SMEs, and freelancers. It automates the invoice reminder process with email and phone follow-ups.

### Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 19 + TypeScript + Vite |
| Styling | Tailwind CSS v4 + Shadcn/UI |
| Backend | Convex (real-time BaaS) |
| AI | Claude 3.5 Haiku (PDF extraction) |
| Auth | @convex-dev/auth |

### Key Features

- PDF invoice import with AI extraction
- Multi-organization/multi-user with roles
- Configurable reminder workflows (email + phone)
- Payment tracking with partial payments
- OAuth email integration (Microsoft, Google)
- Real-time synchronization
